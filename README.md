## News on Map - Real-time news updates mapped to cities

// GEMINI_API_KEY variable for API in enviornment

#Features
News for each city
Simultaneously showcase of news priority
AI analyzes for each city's news

# How it works
1. News is fetched from Google News RSS feeds for each city
2. AI analyzes news importance and assigns priority levels
3. Data is cached and refreshed daily to optimize performance
4. Click on a city to view its latest news headlines
